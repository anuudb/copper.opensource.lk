    Group-Office feature¶

    Group-Office has a modern user interface with an attractive feture list. Group-Office is a PHP based dual license commercial/open source groupware and CRM and DMS product developed by the Dutch company Intermesh. The open source version, Group-Office Community, is licensed under the AGPL, and is available via SourceForge.

    Feature list¶

    Notes¶

    The notes module is a simple note taking modules. The notes module supports:

  1. Links

    2. Group-Office has a very powerful link function. Each item can be linked to another. The link function gives you the ability to organize your data easily. You can for example link:

  2. Sharing

    3. Notes can be shared among groups.

  3. Attaching Files

  4. You can manage several notebooks according to your needs.

    5. Address book¶

    The address book is a very powerful tool to manage contacts and organizations. It’s easy to share address books with other users.

    The address book module supports:

  5. Can link with tasks, calendar events, contacts etc…

  6. Can directly send an email from contact

  7. Can maintain several address books

  8. Export contacts

    9. Bookmark¶

  9. Can maintain a section of URL bookmarks for save and use (anywhere) important sites.

    10. Calendar¶

  10. Send notification mails to participants when events create in calendar

  11. Accepting from mail

    12. Once accepted it can be seen in the event.

  12. Send Events to external users

  13. Allocate a location

  14. Can send a recurrence notification

  15. Can check resource availability through calendar

  16. Can categorize calendar e.g. Dev Calendar, Events Calendar
  17. Enabling reminders

    18. File storage¶

  18. Upload file

  19. Send Download Link through email

  20. Can have an expiration date

  21. Creating new Microsoft word and Open office documentation files

  22. Editing can be done only after download

  23. Can link with tasks, calendar events, contacts etc…

    24. Tasks¶

  24. Allocate Tasks to copper user

  25. Receive Allocated Tasks to copper user

  26. Can link with tasks, calendar events, contacts etc…

  27. Can recurrence notification

  28. Add Reminders

    29. You may add Reminders with time

  29. Can export

    30. Exports available with many formats like, csv, pdf, html etc.
